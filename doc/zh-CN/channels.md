支持的平台：

| 通道 | 准备事项 |
|------|---------|
| [Telegram](channels/telegram.md) | @BotFather 生成 Bot Token |
| [Discord](channels/discord.md) | Bot Token + MESSAGE CONTENT intent |
| [Matrix](channels/matrix.md) | 账号与令牌 |
| [WhatsApp](channels/whatsapp.md) | 扫码登录 |
| [飞书](channels/feishu.md) | App ID + App Secret |
| [QQ](channels/qq.md) | App ID + App Secret |
| [钉钉](channels/dingtalk.md) | App Key + App Secret |
| [Slack](channels/slack.md) | Bot token + App-Level token |
| [Email](channels/email.md) | IMAP/SMTP 账号 |

每个通道的分步配置已拆分为独立页面，点击上表链接查看。
