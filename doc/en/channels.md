Supported chat platforms:

| Channel | Requirements |
|---------|--------------|
| [Telegram](channels/telegram.md) | Bot token from @BotFather |
| [Discord](channels/discord.md) | Bot token + MESSAGE CONTENT intent |
| [Matrix](channels/matrix.md) | Account + tokens |
| [WhatsApp](channels/whatsapp.md) | QR code scan |
| [Feishu](channels/feishu.md) | App ID + App Secret |
| [QQ](channels/qq.md) | App ID + App Secret |
| [DingTalk](channels/dingtalk.md) | App Key + App Secret |
| [Slack](channels/slack.md) | Bot token + App-Level token |
| [Email](channels/email.md) | IMAP/SMTP credentials |

Each channel includes step-by-step setup instructions in its page above.
